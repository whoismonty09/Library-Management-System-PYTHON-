# Library Management System (Python)

## Description
This project is a simple Library Management System built using Python. It allows users to add, view, search, and delete book records using a menu-driven command-line interface. The data is stored in a text file to ensure persistence between program executions.

## Features
- Add new book (Book ID, Title, Author)
- View all books
- Search book by ID
- Delete book record
- Menu-driven command-line interface
- File-based storage (books.txt)

## Technologies Used
- Python
- File Handling
